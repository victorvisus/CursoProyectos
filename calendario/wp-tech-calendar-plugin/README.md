# TechCalendar for WordPress

Este plugin convierte el sistema de gestión de citas técnicas en un componente nativo, profesional e independiente para tu sitio WordPress.

## Instalación

1.  **Copiar carpeta**: Sube la carpeta `tech-calendar-plugin` al directorio `/wp-content/plugins/` de tu servidor.
2.  **Activar**: Ve al panel de **Plugins** en WordPress y activa "TechCalendar - Sistema de Citas".
3.  **Base de Datos**: El plugin creará automáticamente la tabla `wp_tech_appointments` al activarse.

## Uso

Para mostrar el calendario en cualquier página o entrada, utiliza el siguiente shortcode:

```text
[tech_calendar]
```

## Características Principales

- **Independiente de Frameworks**: No requiere Tailwind CSS ni ninguna otra librería externa. Todo el diseño es nativo y está encapsulado para tu tema.
- **Base de Datos Real**: Las citas se almacenan de forma segura en la base de datos de WordPress (MySQL).
- **Comunicación AJAX**: La disponibilidad se consulta al servidor sin necesidad de recargar la página.
- **Diseño Premium**: Interfaz oscura, moderna y responsiva.
- **Validación Avanzada**: Control de errores en tiempo real y seguridad en el lado del servidor.
- **Gestión de Festivos**: Bloqueo automático de fines de semana y festivos locales (Zaragoza).
